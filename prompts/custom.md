# Custom Prompts

## Generate a new Vue component

Create a new Vue.js component with the following structure. The component should include a template, script, and style section. It should also include a basic props definition.

```vue
<template>
  <div>
    <!-- Component HTML -->
  </div>
</template>

<script>
export default {
  name: 'NewComponent',
  props: {
    // Props definition
  }
}
</script>

<style scoped>
/* Component styles */
</style>
```
